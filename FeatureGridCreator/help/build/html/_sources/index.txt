Feature grid aanmaken
---------------------

De plugin kan op dit moment:
- voor een geselecteerde feature (of een hele laag), lijn of polygoon een grid aanmaken
- dit grid kan vierkant of ruitvormig zijn, de afstanden zijn in te stellen
- een grid kan punten of sleuven bevatten
- de x en y afstand van de gridlijnen kunnen worden ingesteld
- de breedte en hoogte van de sleuven kan worden ingesteld
- de aangemaakte features komen in een (tijdelijke) memory layer
- de features kunnen automatisch worden gelabeld door eroverheen te 'hoveren'
- de laatste instellingen worden onthouden
- er kan gekozen worden om alleen BINNEN de polygonen features aan te maken (of juist niet, dan zal er RUIM om de polygoon heen punten worden aangemaakt

Autolabeling
------------

Na het aanmaken van de features, zal die laag automatisch actief zijn, en kan door op de tweede knop te klikken en over de features the bewegen automatisch labels verschijnen (genummerd vanaf 0)


Gebruik
-------

Na het aanmaken van de features (en optioneel labels) dienen de aangemaakte feature te worden 'gecopy/paste' naar de uiteindelijke laag, OF worden opgeslagen als een shapefile.
Dit omdat de aangemaakte feature laag een zogenaamde 'memory'-laag is, dus eigenlijk een tijdelijke laag is.
